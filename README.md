## Installation

```bash
npm install @synth-bridge-service-2/synth-bridge-service-2
```

## Usage

```typescript
import { SynthBridgeService2 } from '@production/synth-bridge-service-2';

const service = new SynthBridgeService2();
await service.initialize();
const result = await service.execute({ data: 'test' });
```

## API

- `initialize()` - Initialize the service
- `execute(input)` - Execute with validation
- `health()` - Health check
- `shutdown()` - Graceful shutdown

## Metrics

- `{name}_requests_total` - Total requests
- `{name}_request_duration_seconds` - Request duration
